## 명세 작성 — `zpicture/minute_stg_spec.md`
전략 하나당:
1. **가설 1줄** (주석 근거)
2. **진입 조건 의사코드** — 봉 수 기준으로 정규화 (예: "MA20(20봉) 상승", "직전 N봉 고점").
   원본 타임프레임(5분/30분) 명시 + 15분봉 적용 시 봉 수 해석 제안 (예: 30분 로직의 20봉 = 10시간
   → 15분봉 40봉으로 볼지 20봉 그대로 볼지 — 원본 의도 기준으로 제안하되 "선택 사항"으로 표시)
3. **청산 조건** (있으면 — 없으면 "명시 없음")
4. **상태 구조** — 단일 순간(edge)인가 장전→발화(armed) 2단계인가, 상태로 뭘 들고 다니는가
5. **주식 전용 요소** — 거래대금 원화 기준, 시가/종가 갭, 장 시작/마감 시각 의존 등
   크립토(24시간, 갭 없음) 적용 시 제거/변환이 필요한 조건 목록
6. **RESTGo 기존 부품 매핑** — 재사용 가능한 조건/트리거 (예: IsMA20RisingStreak, BB 조건들)

마지막: 구현 난이도·측정 가치 기준 추천 착수 순서.
