`zpicture/pullback_scan_s0.json`의 examples를 소스로 사용하라. 신호 재탐지(로직 재구현) 금지 —
아래 지정된 신호의 날짜들을 그대로 마커로 찍기만 한다.

## 차트 요구사항
- 캔들(원본 가격) + MA5/MA20/MA60 (종가 단순이동평균 직접 계산)
- 표시 구간: R box 40봉 전 ~ trigger 40봉 후 (패턴 중심 — 고정 180일 아님)
- 마커 4종 (범례 포함, 구분되는 색):
  - R box (r_date): 저항 박스 — 이 부근에서 "고가는 MA20 위, 종가는 아래" 거부가 보여야 함
  - S box (s_date): 지지(눌림 바닥)
  - arm (arm_date): S box 인지 캔들 (장전)
  - trigger (trigger_date): 양봉 MA20 종가 돌파 (매수 신호) — 가장 크게 강조
- 제목: `<종목> 눌림돌파(streak=0) | trigger <날짜> | r20 <값>%`
- 저장: `zpicture/pullback_example_<번호>_<종목>.png` + chown 1000:1000
- 캔들 데이터는 hannam DB에서 조회 (이전 hns 차트 때 쓴 fetch 방식 재사용 가능)
